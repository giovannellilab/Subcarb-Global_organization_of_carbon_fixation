Data and code relative to the de Pins et al. 2026 manuscript on the global organization of carbon fixation in Earth’s subsurface biosphere. 

This repository contains the following files:

- README.md (this readme file)
- ko_otu_list.csv containign the list fo annotated KO functions for all the used metagenomes
- env_dataset.csv containing the environmental and geochemical data for each metagenome
- tax.csv containing the information and classification of all the KO functions
- de_Pins_et_al_code.ipjnb containing the complete Jupyter Notebook to reproduce the entire analysis and all the figures
